Instructions
____________________________________________________

1. Run WampServer if on Windows, and MampServer if on MacOS.
   MySQL server can also be used on both operating systems. 
   Download one of the above servers if you do not have any of the appropriate servers installed on your laptop.

2. Download MySQLWorkbench or any other Integrated Development Environment (IDE) that can run .sql scripts.
   You can skip this step if you already have an appropriate IDE to run .sql scripts.

3. Run deploy.sql on your IDE.
